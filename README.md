# PlantAR — iOS App

An augmented reality plant identification and learning app built with Swift and ARKit.

## Project Structure

```
PlantAR/
  PlantAR.xcodeproj/    ← Open this in Xcode
  PlantAR/              ← Swift source files and assets
mustard.glb
sunflower.glb
```

## Getting Started

1. Clone the repository
2. Open `PlantAR/PlantAR.xcodeproj` in Xcode
3. Build and run on a physical iOS device (ARKit requires real hardware)

## Features

- AR plant recognition using reference images
- Plant info sheets with detailed descriptions
- Quiz and learning modules
- Teacher dashboard
- Firebase authentication

## Requirements

- Xcode 15+
- iOS 16+
- Physical device with ARKit support
